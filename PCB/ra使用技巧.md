### 文本居中

```
<div align = "center">要居中的文字</div>
```

### 首行缩进2字符

```
&emsp;
```



> 如果要是用的Typora写markdown文章，开头的`Front-matter`应该指定路径；
>
> 如：`typora-root-url: ./最近在看老舍的书籍-我为什么喜欢这类书籍？/`

```
---
title: 最近在看老舍的书籍 我为什么喜欢这类书籍？
tags:
    - 你为什么贫穷
categories:
    - 个人见解
date: 2023-02-18 21:16:57
typora-root-url: ./最近在看老舍的书籍-我为什么喜欢这类书籍？/
---
```
